# liveqr

## About
liveqr은 구매자 중심의 UX를 갖춘 Payment 서비스 프로토 타입입니다.

판매점에서 직접 가격을 입력하면, 사용자는 생성된 QR코드를 이용하여, 결제할 수 있습니다.

해당 Repository에서는 가격 입력 / QR 코드 생성 / 디스플레이 표출 부분을 다룹니다.
